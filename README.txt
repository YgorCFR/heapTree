PARA executar o programa você deve:

instalar o compilador gcc caso não tenha instalado.

e inserir os comandos:

gcc -c arquivo.c (para cada arquivo em c exceto o main.c)

gcc main.c -o main arquivo.o (o mesmo arquivo de cima porém com a extensão .o)

main.exe

os elementos basta digitar P e para tirar o mínimo basta digitar M
